# pauli589.github.io

这是一个免费的 GitHub Pages 个人学术主页模板。

## 如何发布

1. 在 GitHub 创建仓库：`pauli589.github.io`
2. 把本文件夹里的所有文件上传到这个仓库
3. 等待 1-3 分钟
4. 打开 `https://pauli589.github.io`

## 你最需要改的地方

- 把所有页面里的 `你的名字` 改成你的真实姓名或笔名
- 把 `your-email@example.com` 改成你的邮箱
- 把 `assets/hero-study.png` 替换成你的个人照片或保留现在的封面图
- 把新的 PDF 上传到 `files` 文件夹
- 在 `lectures.html` 里把 PDF 链接改成真实文件名

当前模板已经放入了几份本地现有 PDF：

- `files/msd-chapter-3-core.pdf`
- `files/msd-chapter-3-exercises.pdf`
- `files/msd-chapter-4-1-4-2-core.pdf`
- `files/altermagnetism-min-action-plan.pdf`

## PDF 链接示例

如果你上传了：

```text
files/quantum-mechanics.pdf
```

页面里的链接应写成：

```html
<a href="files/quantum-mechanics.pdf">PDF</a>
```

## 以后绑定域名

现在不用买域名。等以后需要时，可以在 GitHub Pages 设置里绑定自己的域名，网站内容不用重做。
